# 🎨 Personal Portfolio Website
A modern, responsive portfolio website built with React and Framer Motion, featuring a sleek dark theme and interactive animations.

## 🌟 Features

### Design & Interactivity
- **Dynamic Navigation**:
  - Smooth scrolling navigation
  - Responsive mobile menu with elastic animation
  - Social media integration
  - Resume download link

### Sections
- **Home**:
  - Interactive profile image with drag animation
  - Tech stack showcase
  - Brief introduction
  
- **About**:
  - Detailed professional background
  - Personal journey
  - Skills and interests

- **Projects**:
  - Featured project showcase
  - Project descriptions
  - Tech stack per project
  - Live demo and code links

- **Experience**:
  - Timeline-based experience layout
  - Detailed work history
  - Key achievements
  - Skills and certifications

- **Contact**:
  - Direct email link
  - Social media connections
  - Professional networking options

### Visual Effects
- **Animations**:
  - Framer Motion transitions
  - Scroll reveal effects
  - Interactive hover states
  - Smooth page load animations

## 🚀 Getting Started

### Prerequisites
- Node.js (18.0 or higher)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/JChurch95/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Environment Setup
Create a `.env` file in the root directory:
```env
VITE_SOME_KEY=your_key_here
```
